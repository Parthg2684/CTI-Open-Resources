## Intelligence Sources & Types

This covers various types of intelligence and sources where information can be gathered.
* **Intelligence Types**: OSINT, HUMINT, IMINT, MASINT
* **Dark Web & Underground Sources**:
    * Telegram channels, such as `moon clould`
    * Black Markets (Credentials & SW)
    * Dark Web Markets (Illegal Goods)
    * Ransomware & Data leak sites, such as the DLS for the CLOP ransomware group.
    * Dark web forums, such as BHF (best hack forum), a Russian hacking community.
